# 전생했더니 이순신장군

조선시대 성(城)을 배경으로 한 **타워 디펜스 + 로그라이크 성장** 게임. 함정과 벽을 배치해 방어선을 구축하는 동시에, 플레이어 캐릭터를 직접 조작해 전투에 참여할 수 있다.

> 📷 *[여기에 타이틀/플레이 스크린샷 삽입]*

## 게임 소개

- **목표**: 5개의 몬스터 웨이브 + 1개의 보스 웨이브로부터 성을 방어
- **배치 시스템**: 그리드 기반 맵 위에 벽(Wall), 가시 함정(Spike), 화염 함정(FireTrap), 폭탄 함정(ExplosiveTrap)을 배치해 적의 경로를 통제하고 피해를 입힘
- **적 유닛**: 근접형(Melee), 원거리형(Ranged), 벽을 무시하고 돌파하는 닌자형(Ninja), 보스 — 유닛마다 벽/함정 대응 방식이 다름
- **플레이어 캐릭터**: WASD로 이동하며 근처 적을 자동으로 조준·공격하는 히어로 유닛 직접 조작
- **성장 시스템**: 웨이브 클리어 후 3가지 업그레이드 중 하나를 선택하는 로그라이크식 성장 (공격력, 공격속도, 사거리, 골드 획득량, 함정 강화 등 총 14종)
- **경제 시스템**: 적 처치·웨이브 클리어로 골드를 획득해 구조물 배치에 사용

## 조작 방법

| 입력 | 동작 |
|---|---|
| `W` `A` `S` `D` | 플레이어 캐릭터 이동 |
| `1` `2` `3` `4` | 벽 / 가시 함정 / 화염 함정 / 폭탄 함정 배치 모드 선택 |
| `5` | 철거 모드 |
| 마우스 좌클릭 | 선택한 구조물 배치 |
| 마우스 우클릭 | 배치 모드 취소 |

## 기술 스택

- **엔진**: Unity 6 (`6000.5.2f1`)
- **언어**: C#
- **주요 시스템**
  - `TileGrid` / `Pathfinder`: 그리드 기반 맵 관리 및 BFS 최단 경로 탐색 (벽 배치 시 자동 재계산)
  - `BuildManager`: 구조물 배치/철거 및 배치 유효성 검증
  - `WaveManager` / `WaveData`: 웨이브 구성 및 스폰 타이밍 관리
  - `Economy`: 골드 획득/소모 관리
  - `UpgradeChoiceManager`: 웨이브 클리어 후 3택 업그레이드 선택
  - `Player` / `Projectile`: 플레이어 이동, 자동 타겟팅, 투사체 발사
  - `Enemy`: 유닛 타입별 이동/전투/벽 대응 로직
